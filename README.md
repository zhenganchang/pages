### 阿里云 Maven 仓库配置

```xml
    <repository>
        <id>aliyun</id>

    </repository>
</repositories>

    <pluginRepository>
        <id>aliyun</id>
        <url>https://maven.aliyun.com/repository/public</url>
    </pluginRepository>
</pluginRepositories>
```
